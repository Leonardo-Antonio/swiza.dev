**MultiDevTools** is a free, open-source collection of browser-based developer tools designed for speed, privacy, and simplicity. It allows developers to perform everyday debugging, formatting, and encoding tasks entirely client-side without sending data to any external server.

🌍 **Available in English and Spanish.**  
🔗 **Live Version:** [www.multidev.tools](https://www.multidev.tools/)

---

## 🚀 Features

- **JSON Formatter & Validator**: Format, validate, and minify your JSON data.
- **Text Diff Checker**: Compare two text sources side-by-side to easily find modifications.
- **JSON to Class Converter**: Instantly transform JSON payloads into TypeScript interfaces or Python classes.
- **JWT Decoder**: Safely decode JSON Web Tokens locally.
- **Base64 Encoder / Decoder**: Translate strings and files back and forth to Base64 formats.
- **Regex Tester**: Test your regular expressions in real-time with highlighted matches.
- **SQL Formatter**: Beautify raw SQL queries across multiple dialects.
- **UUID Generator**: Rapidly generate sequential or random UUIDs in bulk.
- **Log Prettifier**: Clean up unreadable server production logs, with built-in automatic detection and extraction of embedded SQL queries.

## 🛡️ Privacy First

Everything runs **100% locally in your browser**. No databases, no API calls to third parties, and no data tracking. Your tokens, logs, and sensitive codebase strings never leave your machine.

## 🛠️ Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Leonardo-Antonio/swiza.dev.git
   cd toolkitdev
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
